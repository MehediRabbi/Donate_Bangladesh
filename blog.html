What is the Document Object Model (DOM)?

The DOM (Document Object Model) is a programming interface that represents an HTML or XML document as a tree-like structure. It allows scripts to interact with and modify the document's content, structure, and styles dynamically.


How do you select an element from the DOM?

You can select an element using methods such as:
document.getElementById('id')
document.querySelector('selector')
document.querySelectorAll('selector')
document.getElementsByClassName('className')
document.getElementsByTagName('tagName')


What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element to manage events for its child elements. This method enhances performance and efficiently handles events for dynamically added child elements.


How do you manipulate an element's attributes and styles using the DOM?

You can use element.setAttribute('attribute', 'value') to set attributes, and to modify styles, you can use element.style.property or element.classList.
